HTML and HTML5 Cheat Sheet

  1. <Doctype>

    2. Defines the document type

  2. <a>

    3. Defines a hyperlink
    href, rel, type

  3. <address>

    4. Defines an address element

  4. <html>

    5. this tag is used at the beginning and end of an HTML document

  5. <head>

    6. tag that contains information towards the beginning of the document

  6. <title>

    7. tag used for the title of the page, shows up in the browser's title bar

  7. <body>

    8. tag after </head> at beginning and end of all the content that will be shown on the page.

  8. <link>

    9. Used to connect external pages or documents

  9. <style>

    10. includes document style information by defaulting to a CSS file

  10. <script>

    11. includes information or links to external scripts

  11. <h>

    12. levels of headings from 1-6, with 1 being, the bigggest or most important, and 6 being the smallest, or least important
    <h1...h6 >

  12. <div>

    13. tags used to separate a part of the page

  13. <section>

    14. tag used to keep specific parts of the page together as a section

  14. <p>

    15. tag used to organize paragraphs of the text on the page

  15. <strong>

    16. tag used to indicate emphasis on a word by making it appear bold

  16. <b>

    17. Another tag used to create bold text

  17. <i>

    18. tag used to make text italicized

  18. <strike>

    19. tag creates text with a line through it

  19. <abbr>

    20. create abbreviations, while also making the full form available

  20. <img>

    21. image tag used to display images
    src, height, width, align, border

  21. <ol>

    22. creates numbered lists
    li

  22. <ul>

    23. creates a bullet list, unordered list
    li

  23. <form>

    24. creates a form
    type, name

  24. <button>

    25. creates a clickable button
    form, type, name, type, value

  25. <table>

    26. identifes and contains all table related content

  26. <tr>

    27. contains information to be included in a row of the table

  27. <th>

    28. contains information or data in the table header

  28. <td>

    29. contains the data in a single cell

  29. <header>

    30. defines an individual section at the head of the document

  30. <footer>

    31. defines an individual section at the foot of the document

  31. <article>

    32. identifies an article inside the document
    cite

  32. <section>

    33. specifies a specific section of the document

  33. <area>

    34. defines an area inside an image map
    href, rel, type, hreflang

  34. <br>

    35. inserts a single line break

  35. <center>

    36. align text and content

  36. <font>

    37. defines font face, size, and color

  37. <video>

    38. defines a video
    autoplay,controls, height, width, loop, src
